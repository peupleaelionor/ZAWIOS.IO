# ZAWIOS.IO — One-Pager Investisseur

**Infrastructure de signal collectif. Pondéré. Résolvable. Souverain.**

---

## Problème

L'information est fragmentée. Les signaux pertinents — sur l'économie, la société, la technologie, l'Afrique, le climat — se noient dans un flux de bruit permanent. Les plateformes de prédiction existantes sont anglophones, financiarisées et déconnectées des réalités du monde francophone et africain. Les analyses sérieuses n'ont pas de lieu d'ancrage, pas de pondération, pas de résolution.

Résultat : les décideurs, analystes et observateurs naviguent à vue.

---

## Solution

**ZAWIOS est une infrastructure de signal collectif** : une plateforme où des analystes humains votent OUI / NON / NEUTRE sur des questions précises et résolvables, avec un système de pondération par fiabilité historique.

Chaque signal est :
- **calibré** — pondéré par le score de précision passé du contributeur
- **résolvable** — ancré dans une date de vérification et une source
- **nuancé** — les votes portent des arguments, pas seulement une position

Le résultat : un feed d'intelligence collective fiable, contrasté, francophone.

---

## Pourquoi maintenant

L'IA générative produit du contenu en masse mais ne garantit ni la fiabilité ni la diversité de perspective. Elle crée plus de bruit que jamais. Le besoin de **filtres humains calibrés** devient critique.

ZAWIOS positionne l'intelligence humaine collective comme la couche de confiance que l'IA ne peut pas remplacer.

## Produit

| Fonctionnalité | Description |
|---|---|
| **Feed calibré** | Signaux triés par qualité prédictive, pas par popularité |
| **HOTD** | Hot of the Day — signal du jour mis en avant, pic d'engagement communautaire |
| **Leaderboard de précision** | Classement des contributeurs par score de calibration (Brier score) |
| **Nuances** | Les votants peuvent ajouter des conditions, des réserves, des arguments |
| **Anti-spam** | Système de trust weights, détection de comportements anormaux |
| **7 agents IA** | Agents de vérification, de résumé, de résolution, de modération |
| **50+ signaux actifs** | Société, économie, tech, Afrique, climat |

## Modèle économique

ZAWIOS n'est pas un investissement financier pour ses contributeurs. C'est de **l'infrastructure**.

| Source | Description | Horizon |
|---|---|---|
| **Contribution libre** | Accès gratuit, vote, nuances | Lancement |
| **Premium analytics** | Feed avancé, historique, alertes, export | M+3 |
| **API institutionnelle** | Accès machine pour médias, think tanks, entreprises | M+9 |
| **Partenariats data** | Organisations souhaitant intégrer l'intelligence collective | M+12 |

Aucun rendement financier pour les contributeurs. La valeur est la réputation, la visibilité analytique, et l'impact.

## Go-to-market (60 jours)

1. **Ciblage initial** : analystes africains et francophones (Sénégal, Côte d'Ivoire, Maroc, France, Belgique, Canada) — communautés déjà actives sur X/Twitter et LinkedIn
2. **Partage organique** : chaque signal HOTD est partageable avec carte visuelle Royal Signal Blue
3. **Invitations** : système de cooptation — les contributeurs de confiance invitent, le réseau se qualifie
4. **Partenariats éditoriaux** : médias francophones, newsletters, podcasts d'analyse

## Equipe

- **Product-led** : construction guidée par les usages réels, itération rapide
- **Indépendance éditoriale** : neutralité stricte sur les signaux, aucune influence externe sur les votes
- **Architecture ouverte** : transparence des règles de pondération et des critères de résolution
- **Francophonie au coeur** : langue, culture, géographie africaine comme axes stratégiques premiers
